Display

Display object are helpers to manipulate visual elements built on DOM like images, videos, sprites.

Example :

{
  "url": "./videos/video.mp4"
}

video #

name type prop description
url string required
 The url of the video file.

Example : 
"./videos/video.mp4"

sprite #

name type prop description
url string required
 The url of the sprite map image file.

Example : 
"./sprites/sprite-map.png"
frames string required
 The url of the sprite frames json file.

Example : 
"./sprites/sprite-map.json"
frameRate number default: 30
 The default number of frame per second to use for all animations. That modifies the speed of the animation.

Example : 
24
frame number The starting frame index

Example : 
5
paused boolean If paused is set to true, the sprite will be paused at start.

Example : 
true
loop boolean default: true
 If loop is set to true, the sprite animations will loop by default.

Example : 
false
animation string The name of the default animation.

Example : 
"animation-0"
animations array<object> Array of animations configurations.

sprite-animation #

name type prop description
name string required
 The name of the animation.

Example : 
"animation-0"
start number The index of the starting frame.

Example : 
0
end number The index of the ending frame.

Example : 
24
loop boolean default: true
 If loop is set to true, the sprite animation will loop

Example : 
false
frameRate number default: 30
 The number of frame per second to use for this animation. That modifies the speed of the animation.

Example : 
24
