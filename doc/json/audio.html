Audio

The global audio volume configuration, valid for either video and audio track. Volume can be changed for a video and an audio track, but take into account that this global audio config has still an effect on those others audio configuration.

Example :

{
  "enabled": true,
  "volume": {
    "default": 1,
    "max": 1
  }
}

audio #

name type prop description
enabled boolean default: true
 Is the audio interface enabled ?

Example : 
true
volume object The configuration of the volume of the audio.

audio-volume #

name type prop description
default number default: 1
max: 1
 The default value of the volume.

Example : 
1
max number default: 1
max: 1
 The maximum value of the volume.

Example : 
1
