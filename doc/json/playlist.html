Playlist

Playlists are composed of audio tracks to play. Each track is independant of each playlist, and thus a track can be used for multiple playlists.

Example :

{
  "enabled": true,
  "volume": {
    "default": 1,
    "max": 1
  },
  "default": {},
  "lists": [
    {
      "uid": "playlist-0",
      "name": "Killer Playlist",
      "default": {},
      "autoPlay": true,
      "loop": true,
      "volume": {
        "default": 1,
        "max": 1
      },
      "tracks": [
        "track-0",
        "track-1"
      ]
    }
  ],
  "tracks": [
    {
      "uid": "track-0",
      "name": "My awesome song",
      "author": "John Doo",
      "url": "audio/track-0.mp3",
      "loop": false
    }
  ]
}

playlist #

name type prop description
enabled boolean default: true
 Is it enabled ?

Example : 
true
volume object The configuration of the volume of the audio.
default
  • string
  • number
    		•  The default playlist UID
    lists array<object> The list of playlists. Each member is an instance of a playlist.
    tracks array<object> The list of tracks. Each member is an instance of a track.

    playlist-config #

    name type prop description
    uid string required
    		 The UID of the playlist.

    Example : 
    "playlist-0"
    tracks array<string> required
    		 The list of tracks UID associated to the playlist.
    name string The name of the playlist.

    Example : 
    "Killer Playlist"
    default
  • string
  • number
    		•  The default track UID of the playlist.
    autoPlay boolean default: true
    		 This sets whether or not the playlist starts automatically.

    Example : 
    true
    loop boolean default: true
    		 This sets whether or not the playlist repeats after the play of the last track.

    Example : 
    true
    volume object The configuration of the volume of the audio.

    playlist-track #

    name type prop description
    uid string required
    		 The UID of the track.

    Example : 
    "track-0"
    url string required
    		 The file URL of the track.

    Example : 
    "audio/track-0.mp3"
    name string The name of the track.

    Example : 
    "My awesome song"
    author string The author of the track.

    Example : 
    "John Doo"
    loop boolean This sets whether or not the track repeats after a script tells it to play.

    Example : 
    false
