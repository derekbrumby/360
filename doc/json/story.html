story #

name type prop description
uid string required
 The UID of the story.

Example : 
"story-0"
name string required
 The name of the story or its i18n key.

Example : 
"My Story"
slug string required
 The slug of the story or its i18n key.

Example : 
"my-story"
description string required
 The description of the story or its i18n key.

Example : 
"This is a super story about awesome things"
default string required
 The UID of the default scene to load on start.

Example : 
"scene-0"
thumbnails array<object> The list of thumbnails in the story. Each member is an instance of a thumbnail.
scenes array<object> The list of scenes in the story. Each member is an instance of a scene.
groups array<object> The list of groups in the story. Each member is an instance of a group.
events object An instance of the story events configuration.
