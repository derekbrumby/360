Camera

The camera can have its initial position set, and have limitations in its movement. This configuration can be set globally at the root, but a scene can also have a specific configuration for its camera.

Example :

{
  "parallax": 0.2,
  "keep": "true",
  "yaw": {
    "default": 45,
    "min": 0,
    "max": 90
  },
  "pitch": {
    "default": 45,
    "min": 0,
    "max": 90
  },
  "roll": {
    "default": 45,
    "min": 0,
    "max": 90
  },
  "fov": {
    "default": 45,
    "min": 0,
    "max": 90
  },
  "gaze": {
    "delay": 2000,
    "cursor": {
      "innerRadius": 0.02,
      "outerRadius": 0.04,
      "color": {},
      "opacity": 0.5
    },
    "progress": {
      "innerRadius": 0.02,
      "outerRadius": 0.04,
      "color": {},
      "opacity": 0.5
    }
  }
}

camera #

name type prop description
parallax number max: 1
 Parallax factor.

Example : 
0.2
keep boolean Does the camera have to keep its previous configuration on scene load?

Example : 
"true"
yaw object An instance of a camera value.
pitch object An instance of a camera value.
roll object An instance of a camera value.
fov object An instance of a camera value.
gaze object Camera gaze cursor configuration

camera-value #

name type prop description
default number The default value.

Example : 
45
min number The minimum value.

Example : 
0
max number The maximum value

Example : 
90

camera-gaze #

name type prop description
delay number The time needed to validate a gaze interaction.

Example : 
2000
cursor object An instance of a camera gaze ring geometry and material configuration.
progress object An instance of a camera gaze ring geometry and material configuration.

camera-gaze-ring #

name type prop description
innerRadius number default: 0.02
 The inner radius of the ring.

Example : 
0.02
outerRadius number default: 0.04
 The outer radius of the ring.

Example : 
0.04
color
  • string
  • number
    		•  The hexadecimal color code of the ring.
    opacity number max: 1
    		 The opacity of the ring.
