  "target": {},

    "name": "lookAt",

      0,

  }

action #

name type prop description
uid string required
 The UID of the action.

Example : 
"action-more-awesome"
target
  • string
  • object
    		•  required
    		 The UID of the target of the action, the list of accessor to get the target from the Viewer object, or a combination of both.
    method object An instance of an action method.
    property object An instance of an action property.

    action-target #

    name type prop description
    identifier string The UID of the target.

    Example : 
    "hotspot-3d"
    accessor string The list of accessor (chained with a dot) to get the target from the UID object.

    Example : 
    "animation"

    action-method #

    name type prop description
    name string The name of the method.

    Example : 
    "lookAt"
    args array<> The arguments array you want to pass to this method.

    action-property #

    name type prop description
    name string The name of the property.

    Example : 
    "fov"
    value The value you want to set to this property, can be of any valid JSON type.

    Example : 
    10
    operation string The kind of operation you want to apply to this property. Can be one of the following:
    • set - set the current value to the passed value
    • add - add the passed value to the current value
    • substract - substract the passed value to the current value
    • multiply - multiply the current value by the passed value
    • divide - divide the current value by the passed value
    • toggle - toggle the current value to false if it was true, or the other way around (only work if the property is of boolean type)


    Example : 
    "add"
