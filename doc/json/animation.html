Animation

Animation can be found in the director's cut and in hotspots.

Example :

{
  "default": "easeInQuad",
  "start": 5000
}

animation-track-easing #

name type prop description
default string The default easing to apply; available values are :
  • linear
  • easeInSine
  • easeInOutSine
  • easeOutSine
  • easeInQuad
  • easeInOutQuad
  • easeOutQuad
  • easeInCubic
  • easeInOutCubic
  • easeOutCubic
  • easeOutBounce
  • easeInBounce
Default is linear.

Example : 
"easeInQuad"
start number The time (in ms) to reach the first position of the track from the current position of the camera.

Example : 
5000
