Group

A group is composed of multiple scenes, so scenes can be regrouped together if it is wanted.

Example :

{
  "uid": "group-0",
  "children": [
    "scene-0",
    "scene-1"
  ],
  "default": 1,
  "name": "My Group",
  "slug": "my-group",
  "description": "This a group with two awesome scenes"
}

group #

name type prop description
uid string required
 The UID of the group.

Example : 
"group-0"
children array<string> The UID of scenes included in this group.
default string The default child of the group.

Example : 
1
name string The name of the group or its i18n key.

Example : 
"My Group"
slug string The slug name of the group or its i18n key.

Example : 
"my-group"
description string The description of the group or its i18n key.

Example : 
"This a group with two awesome scenes"
