Sound

A sound can be really simple, just loaded on the scene, or more complex: only perceptible in a specific corner of the scene, ambisonic, etc...

Example :

{
  "uid": "sound-0",
  "type": "stereo",
  "source": {
    "url": "audio/scene-0-hotspot.mp3",
    "target": "video-0"
  },
  "options": {
    "volume": {},
    "loop": true,
    "autoPlay": true,
    "startTime": 0,
    "range": 180
  }
}

sound #

name type prop description
uid string required
 The UID of the sound.

Example : 
"sound-0"
type string default: stereo
 What is the type of the sound :
  • stereo
  • 3d
  • ambisonic


Example : 
"stereo"
source object An instance of sound source.
options object An instance of sound options.

sound-source #

name type prop description
url string URL of the sound: can be either a local sound or one on a distant server, as long as it is a valid sound file.

Example : 
"audio/scene-0-hotspot.mp3"
target string The UID of the sound source target: can be either a video or an audio media.

Example : 
"video-0"

sound-options #

name type prop description
volume
  • number
  • object
    		•  The volume of the sound.
    loop boolean Does the sound need to be looped automatically at the end of playing ?

    Example : 
    true
    autoPlay boolean Does the sound need to be played automatically at start ?

    Example : 
    true
    startTime number The starting time of the sound in milliseconds, if it needs to start after a certain time

    Example : 
    0
    range number default: 360
    min: 1
    max: 360
    		 The range of the sound in degrees, where sound can be played from it's position before it can't be heared anymore. From 1 to 360.

    Example : 
    180

    sound-volume #

    name type prop description
    min number max: 1
    		 The minimum volume of the sound.

    Example : 
    0.2
    max number default: 1
    max: 1
    		 The maximum volume of the sound.

    Example : 
    0.8
