Hotspot

A hotspot can only be a 3D hotspot for now. A 3D hotspot is a hotspot attached to the sphere containing the 360º media. It can have a visual representation, from an image showing the hotspot, but also a sound, that will be an ambisonic one. The hotspot can stay just like this, or it can be bind to an action, to give an interaction.

hotspot #

name type prop description
uid string The UID of the hotspot.

Example : 
"hotspot-0"
tags array<string> The list of tags associated to the hotspot.
type string default: 3d
 The type of the hotspot. Can be one of :
  • 3d


Example : 
"3d"
visible boolean default: true
 Is the hotspot visible ?

Example : 
true
interactive boolean default: true
 Is this hotspot is interactive and eligible to raycast

Example : 
true
cursor string default: pointer
 The cursor to display when the pointer is over the hotspot. Any valid css cursor value can be set.

Example : 
"crosshair"
fx string The UID of the special effect (FX) to apply to the hotspot.

Example : 
"fx-0"
facingCenter boolean Is the hotspot (for a flat 3D one) facing the center of the scene (the camera) constantly ? Its rotation will be determined given this facing.

Example : 
true
autoScale boolean Does the hotspot auto changes its scale when the fov change?
transform object An instance of a hotspot transform.
geometry object An instance of a hotspot geometry configuration.
animation object An instance of a hotspot animation.
material object An instance of a hotspot material.
sound object An instance of a sound.
dom object An instance of a hotspot dom.
events object An instance of the hotspot events object.
states object An instance of the hotspot states object.

hotspot-geometry #

name type prop description
type string default: plane
 The type of geometry to create. There are all derivate of THREE.JS geometries. Available values are :
  • plane for PlaneBufferGeometry
  • box for BoxBufferGeometry
  • sphere for SphereBufferGeometry
  • cylinder for CylinderBufferGeometry
  • shape for ShapeBufferGeometry


Example : 
"plane"
offset object An instance of a hotspot geometry offset.
options object The options of the geometry, specific and different for each geometry. See the list of available geometry above and their list of parameters.

Example : 
{
  "points": [
    [
      0,
      0
    ],
    [
      10,
      0
    ],
    [
      5,
      4
    ]
  ]
}

    Hotspot Animation

    Each hotspot can be animated. The animation can begin at any time specified by the user, for example by a click or automatically after 10 seconds. This section is placed at the root of the JSON configuration.

    Example :

    {
  "tracks": [
    {
      "uid": "track-0",
      "name": "Sea to beach",
      "description": "This is the track going from the sea to the beach.",
      "smooth": true,
      "cancelRoll": false,
      "easing": {
        "default": "easeInQuad",
        "start": 5000
      },
      "keyframes": [
        {
          "time": 20,
          "position": {
            "radius": 200,
            "theta": 90,
            "phi": 30,
            "x": 240,
            "y": 350,
            "z": 480
          },
          "rotation": {
            "x": 8,
            "y": 3,
            "z": 5
          },
          "scale": {
            "x": 8,
            "y": 3,
            "z": 5
          }
        }
      ]
    }
  ]
}

    hotspot-anim #

    name type prop description
    tracks array<object> A list of the tracks that can be used for animating hotspot-anim. Each member is an instance of a hotspot track.

    hotspot-track #

    name type prop description
    uid string The UID of the track.

    Example : 
    "track-0"
    name string The name of the track.

    Example : 
    "Sea to beach"
    description string The description of the track.

    Example : 
    "This is the track going from the sea to the beach."
    smooth boolean Is the track smooth ?

    Example :

    cancelRoll boolean Is the roll cancelled ?

    Example :

    easing object An instance of an animation track.
    keyframes array<object> A list of the keyframes composing the track. Each member is an instance of a hotspot track keyframe.
