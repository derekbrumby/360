Post Processing

It is possible to apply special effects (FX) on a scene. Each object describing the configuration of a special effect is specific to the latter, as they don't have the same parameters. The special effect is then linked to the scene by specifying its UID to the scene.

Example :

{
  "fxSets": [
    {
      "uid": "fx-set-0",
      "set": [
        {
          "type": "HorizontalBlurShader",
          "uid": "fx-horizontalblur-0",
          "params": {
            "h": 0.0001
          }
        },
        {
          "type": "VerticalBlurShader",
          "uid": "fx-verticalblur-0",
          "params": {
            "v": 0.001
          }
        }
      ]
    }
  ]
}

postprocessing #

name type prop description
fxSets array<object> The list of special effect sets. Each member is an instance of a special effect set.

postprocessing-set #

name type prop description
uid string required
 The UID of the set.

Example : 
"fx-set-0"
set array<object> required
 The list of special effect composing the set. Each member is an instance of a special effect.

postprocessing-fx #

name type prop description
uid string required
 The UID of the special effect.

Example : 
"fx-0"
type string required
 The type of the special effet.

Example : 
"HorizontalBlurShader"
params object The list of parameters to pass to the shader doing the special effect.
