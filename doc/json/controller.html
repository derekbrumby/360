Controllers

Controllers can be bound to ForgeJS: by example, the keyboard and the mouse. By default, some bindings exists, like the arrow keys to move in the scene, and the mouse movement as well.

Example :

{
  "enabled": true,
  "instances": [
    {
      "uid": "controller-0",
      "type": "pointer",
      "enabled": true,
      "options": {
        "fullscreen": true,
        "orientation": {
          "drag": false,
          "hardness": 0.6,
          "damping": 0.15,
          "velocityMax": 300,
          "invert": {}
        },
        "zoom": {
          "hardness": 5,
          "invert": false,
          "toPointer": false
        }
      }
    }
  ]
}

controller #

name type prop description
enabled boolean default: true
 Enables or not all the controllers.

Example : 
true
instances array<object> The list of each controller. Each member is an instance of a controller.

controller-instance #

name type prop description
uid string required
 UID of the controller instance.

Example : 
"controller-0"
type string required
 Type of the controller to instanciate. Can either be :
  • pointer
  • keyboard
  • gyroscope
  • gamepad


Example : 
"pointer"
enabled boolean required
 Is this controller enabled at its creation ?

Example : 
true

    controller-keyboard #

    name type prop description
    orientation object An instance of a controller orientation configuration.
    zoom object An instance of a controller zoom configuration.
    default boolean default: true
    		 Loads the default controller bindings if not set to false.

    Example : 
    true
    bindings array<object> Array of custom keyboard key bindings.
