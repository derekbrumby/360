Director

The director's cut allows to get an automatic navigation inside a 360º media, be it a video or an image. By specifying the different keyframes, the player will navigate through the media, doing interpolation between each keyframe, to display a smooth navigation.

Example :

{
  "animation": {
    "enabled": true,
    "delay": 2000,
    "smooth": true,
    "stoppable": false,
    "idleTime": 3000,
    "random": false,
    "loop": false,
    "tracks": [
      "track-0",
      "track-2"
    ]
  },
  "tracks": [
    {
      "uid": "track-0",
      "name": "Sea to beach",
      "description": "This is the track going from the sea to the beach.",
      "smooth": true,
      "cancelRoll": false,
      "easing": {
        "default": "easeInQuad",
        "start": 5000
      },
      "keyframes": [
        {
          "time": 20,
          "data": {
            "yaw": 10,
            "pitch": 20,
            "roll": -50,
            "fov": 270
          }
        }
      ]
    }
  ]
}

director #

name type prop description
animation object An instance of a director animation.
tracks array<object> A list of the tracks composing the director's cut. Each member is an instance of a director track.

director-animation #

name type prop description
tracks array<string> required
 The list of tracks UID ot execute. The tracks are executed in the order they are written in this array.
enabled boolean Is the animation enabled ?

Example : 
true
delay number Is there a timer to wait for start ? If yes, specify a time in ms.

Example : 
2000
smooth boolean Does the animation have a smooth start from the actual point the camera is looking at.

Example : 
true
stoppable boolean Can the user stop the animation ?

Example : 
false
idleTime number Is there a time to wait for start after user interaction ? If yes, specify a time in ms. If the time is negative, it is considered that the animation never resume.

Example : 
3000
random boolean Are the tracks played randomly ?

Example : 
false
loop boolean Does the animation loop ?

Example : 
false

director-track #

name type prop description
uid string The UID of the track.

Example : 
"track-0"
name string The name of the track.

Example : 
"Sea to beach"
description string The description of the track.

Example : 
"This is the track going from the sea to the beach."
smooth boolean Is the track smooth ?

Example : 
true
cancelRoll boolean Is the roll cancelled ?

Example : 
false
easing object An instance of an animation track.
keyframes array<object> A list of the keyframes composing the track. Each member is an instance of a director track keyframe.

director-track-keyframe #

name type prop description
time number The time to trigger the action.

Example : 
20
data object An instance of a director track keyframe data.

director-track-keyframe-data #

name type prop description
yaw number The yaw of the camera.

Example : 
10
pitch number The pitch of the camera.

Example : 
20
roll number The roll of the camera.

Example : 
-50
fov number The field of view of the camera.

Example : 
270
